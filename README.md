# meu-repositorio-de-autenticacoes-DIO
Repositorio de estudos GIT/GITHUB

Neste repositorio irei testar todos os processos relacionados a GIT no GITHUB, Para que assim eu desenvolva minhas habilidades correlacionadas a ferramenta e assim consiga tornar algo mais natural em meu uso, inclusive para coisas pessoais, dessa forma treinando e me capacitando para me tornar fluente na ferramentas


- primeiro processo foi autenticação com email e senha;
- segundo processo envolvendo autenticação via token;
- terceiro processo envolvendo autenticação com Chave SSH;
